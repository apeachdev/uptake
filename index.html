Emerging Circle

A Financial Debt Lending Platform.

The Emerging Circle Platform was built in partnership with Maek.co.uk. We evaluated the problems in the financial sector and built a robust platform for evaluating small and medium sized business and their eligibility for a loan. We created a comprehensive credit score mechanism as well as a loan marketplace. Finally, we built an integration system with a Restful API for deep integration with 3rd party financial institutions and banks.

Technology Stack: Python, Django, JQUERY, AWS, S3 Bucket, LESS CSS.

Rotation

Social Analytics and Chat Amalgamation App for Android

Rotation is an Android app that was first built as a social analytics tool and then pivoted into a chat amalgamation app. It takes any chat from multiple sources and places those messages into a single feed. Furthermore, it allows a user to take any message and turn it into a todo task. Needless to say the complexity on the backend and server side are immense and these limitations caused it to only be available for Android.

Technology Stack: Native Android Development in Java with a Python/Django/PostreSQL Serverside stack
